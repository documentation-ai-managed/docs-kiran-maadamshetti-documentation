<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added PDF and HTML export options for full docsites.
- Introduced search indexing for faster navigation across 100+ pages.

## Improvements

- Optimized rendering speed by 40% for sites with heavy MDX content.
- Enhanced sidebar navigation with collapsible sections.

## Bug Fixes

- Fixed sync issues with external Git repositories.
- Resolved mobile responsiveness for embedded code blocks.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Real-time collaboration: Multiple users edit docs simultaneously.
- Brand customization: Set custom colors like `#3B82F6` and logos.

## Improvements

- Upgraded MDX parsing for better component support.

## Bug Fixes

- Corrected frontmatter validation errors.
- Patched preview pane refresh loops.

</Update>

</Update>
